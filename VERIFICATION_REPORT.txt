🆘 COMMON ISSUES & SOLUTIONS

Issue: "command not found: jupyter"
Fix:   Make sure environment is activated
       $ source .venv/bin/activate
       Terminal should show (.venv) prefix

Issue: "ModuleNotFoundError: No module named 'tensorflow'"
Fix:   Environment not activated
       Verify: echo $VIRTUAL_ENV
       Should show: /Volumes/MacSSD/.../brain-ai/.venv

Issue: "Permission denied" on setup script
Fix:   $ chmod +x setup_environment.sh
       $ ./setup_environment.sh

Issue: Want to recreate environment
Fix:   $ rm -rf .venv
       $ ./setup_environment.sh
       Or manually: $ source setup_environment.sh

For more troubleshooting, see ENVIRONMENT.md#troubleshooting
