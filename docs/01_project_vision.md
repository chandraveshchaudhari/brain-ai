# Brain-AI: Multimodal LLM-Driven AutoML Framework

## Vision

Brain-AI is a **multimodal AutoML framework** powered by:

* LLMs
* Multiple AutoML libraries
* Tool-based architecture
* Human-in-the-loop learning
* Reproducible pipelines

Goal:

Allow **non-technical users** to perform multimodal learning.

---

## Key Features

* Multimodal Learning
* LLM Pipeline Generation
* Multiple AutoML Libraries
* Tool-based Architecture
* UI for Non-technical Users
* CLI Support
* VS Code Integration
* Reproducible Pipelines
